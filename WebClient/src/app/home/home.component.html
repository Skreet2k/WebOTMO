Я сука смог. Как спрошу я у себя? Вот.

1. Хостинг для Angular на firebase.google.com
2. Хостинг для .net core 2.0 на heroky.com
3. Домен на reg.ru

0. Собираем проект Angular в прод
$ ng build --prod

1. Регистрируемся на firebase.google.com

2. В Console добавляем проект с любым названием в России

3. Переходим в Hosting.
Далее по инструкции устанавливаем CLI
$ npm install -g firebase-tools
Логинимся
$ firebase login
Инициируем проект в папке с проектом
$ firebase init
Деплоим
$ firebase deploy

4. Подключаем домен
Добавляем в TXT домена в reg.ru запись, которую потребует firebase
Добавляем A записи, которые потруем firebase и удалем другие.
Готово

0. Билдим проект .net core в релиз
$ dotnet build --configuration Release

1. Регистрируемся на heroku.com

2. В dashboard добавляем приложение.

3. Deploy в heroku
Выбираем Deployment method Heroku git
Устанавливаем CLI
$ npm install -g heroku-cli
Выбираем билдпак для .net core 2.0
$ heroku buildpacks:set https://github.com/jincod/dotnetcore-buildpack
Логинимся по почте и паролю
$ heroku login
В дерриктории с проектом инициализиуем git
$ git init
Подключаем git к Heroku
$ heroku git:remote -a "app name"
Паблишим
$ git add .
$ git commit -am "commit name"
$ git push heroku master

Готово!
